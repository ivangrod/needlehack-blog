0.67: Mastodon, Tahoma switches, Nanoleaf Aurora Light Panels

Paulus Schoutsen April 13, 2018
  • Core
  • Release-Notes
 Comments

Release 0.67 is here! In case you missed the announcement from yesterday, Ubiquiti is paying me to write these release notes. Ooooooh yeah. With more time on my hands I‚Äôm going to be focusing on major improvements that we have been postponing for a while, first up will be a user system.

This release includes a security fix. The error log was accessible via the API without requiring authentication in previous releases. Although not a leak on its own, combined with a faulty component that would log sensitive information to the error log could lead to that being exposed (we don‚Äôt know of any component that does this). Thanks to Matt Hamilton for disclosing this issue to us.
