Got Some questions?

I am already using Home Assistant. Is this for me?

If you are not familiar with tinkering with Raspberry Pi, or if you want a well-supported and dedicated piece of hardware to run Home Assistant, Home Assistant Green is for you.

If you are still looking for a present for friends and family, this would be perfect, too!

What is the difference between Home Assistant Yellow and Home Assistant Green?

The most obvious difference is that Home Assistant Green is a plug-and-play system. It is designed to be used as-is. Home Assistant Yellow is designed to be extended by more RAM or more storage by adding an NVMe SSD. But extending also means you have to open the housing, install the additional hardware, and go through more complex installation steps.

The other big difference is that Yellow comes with support for Zigbee 3.0 and Matter-over-Thread. But Green can also use these protocols by adding a Connect ZBT-1.

Do I need a Home Assistant Connect ZBT-1?

It is not strictly required, as most smart home devices and hubs work with Home Assistant Green out-of-the-box, but you should confirm compatibility. For the best experience, pair Home Assistant Connect ZBT-1 with Home Assistant Green, making a single hub solution for your Zigbee or Thread powered smart home.

Home Assistant Connect ZBT-1 allows Zigbee or Thread* devices to connect directly to Home Assistant Green.

* Thread support is currently under development.

Will Nabu Casa keep supporting Home Assistant Green?

Yes! We are committed to keep supporting all platforms that are currently available.

Will Nabu Casa keep supporting Home Assistant Yellow?

Absolutely! We believe in Home Assistant Yellow! Depending on who you are and what your smart home looks like, Home Assistant Yellow might be the better choice for you.

Will Home Assistant drop support for the Raspberry Pi?

Nope! We are committed to keep supporting all platforms that are currently available.
